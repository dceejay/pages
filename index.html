Running Node-RED on Android

The Termux app in the android app store makes it easy to run Node-RED on android devices. You can get it here.

Install it, and run it. Then at the prompt type

apt update
apt upgrade
apt install coreutils nano nodejs
npm install -g --unsafe-perm node-red
node-red

Then you can point a browser to localhost:1880

Notes

  • You can npm install other node-red nodes such as node-red-dashboard.
  • The volume-down key is the ctrl key - so vol-down-c can be used to "break" a running app.
  • The instructions above also install the nano editor, as I find it useful for hacking files.
  • For a discussion on autostarting - see this thread on the Termux Github project.
  • Likewise here is a list of other useful Termux packages.
